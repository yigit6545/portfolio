Vue 3 Composition API ile Modern Component Geliştirme

Frontend Developer Senior Developer
Vue 3 Composition API Components

Vue 3'ün Composition API'si, component'ları daha modüler ve yeniden kullanılabilir hale getiriyor. Bu yazıda Composition API'nin avantajlarını ve pratik kullanım örneklerini inceleyeceğiz.

Composition API Nedir?

Composition API, Vue 3'te tanıtılan yeni bir component yazma yaklaşımıdır. Options API'nin aksine, logic'i fonksiyonlar halinde organize eder ve daha iyi kod organizasyonu sağlar.

Composition API Örneği 
<template>
  <div>
    <h1>{{ title }}</h1>
    <p>Count: {{ count }}</p>
    <button @click="increment">Increment</button>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const title = ref('Vue 3 Composition API')
const count = ref(0)

const increment = () => {
  count.value++
}

const doubleCount = computed(() => count.value * 2)
</script>

Composables ile Logic Paylaşımı

Composables, logic'i yeniden kullanılabilir fonksiyonlar halinde organize etmenizi sağlar. Bu sayede aynı logic'i farklı component'larda kullanabilirsiniz.

useCounter Composable 
// composables/useCounter.js
import { ref, computed } from 'vue'

export function useCounter(initialValue = 0) {
  const count = ref(initialValue)
  
  const increment = () => count.value++
  const decrement = () => count.value--
  const reset = () => count.value = initialValue
  
  const doubleCount = computed(() => count.value * 2)
  
  return {
    count,
    increment,
    decrement,
    reset,
    doubleCount
  }
}

Reactivity ile Gelişmiş State Yönetimi

Composition API ile reactive state'i daha kolay yönetebilirsiniz. ref, reactive ve computed gibi fonksiyonlar ile güçlü reactivity sistemi.

Önemli Not

Composition API, Vue 3'te varsayılan olarak mevcut ancak Options API hala desteklenmektedir. Her iki yaklaşımı da aynı projede kullanabilirsiniz.

Sonuç

Vue 3 Composition API, modern component geliştirme için güçlü bir araçtır. Daha iyi kod organizasyonu, logic paylaşımı ve TypeScript desteği sağlar.

Özet

  • Composition API ile daha modüler component'lar
  • Composables ile logic paylaşımı
  • Gelişmiş reactivity sistemi
  • TypeScript desteği
  • Daha iyi kod organizasyonu
