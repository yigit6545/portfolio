CSS Grid vs Flexbox: Hangi Durumda Hangisini Kullanmalı?

Frontend Developer Senior Developer
CSS Grid Flexbox Layout

CSS Grid ve Flexbox, modern web tasarımında en güçlü layout araçlarıdır. Ancak hangi durumda hangisini kullanmanız gerektiğini bilmek önemlidir.

CSS Grid Nedir?

CSS Grid, 2D layout sistemi sağlar. Hem satır hem de sütun boyunca elementleri yerleştirebilirsiniz. Grid, karmaşık layout'lar için idealdir.

CSS Grid Örneği 
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 1fr auto;
  gap: 20px;
  height: 100vh;
}

.header { grid-column: 1 / -1; }
.sidebar { grid-column: 1; }
.main { grid-column: 2; }
.aside { grid-column: 3; }
.footer { grid-column: 1 / -1; }

Flexbox Nedir?

Flexbox, 1D layout sistemi sağlar. Elementleri tek bir eksen boyunca (satır veya sütun) yerleştirir. Basit layout'lar ve component'lar için idealdir.

Flexbox Örneği 
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 20px;
}

.flex-item {
  flex: 1;
}

.flex-item:nth-child(2) {
  flex: 2;
}

Ne Zaman CSS Grid Kullanmalı?

CSS Grid'i şu durumlarda kullanın:

CSS Grid Kullanım Senaryoları

  • 2D layout'lar (satır ve sütun)
  • Karmaşık sayfa yapıları
  • Elementlerin kesişen alanları
  • Responsive grid sistemleri
  • Card layout'ları

Ne Zaman Flexbox Kullanmalı?

Flexbox'ı şu durumlarda kullanın:

Flexbox Kullanım Senaryoları

  • 1D layout'lar (tek eksen)
  • Navigation menüleri
  • Button grupları
  • Form elementleri
  • Basit component'lar

Birlikte Kullanım

Grid ve Flexbox'ı birlikte kullanabilirsiniz. Grid container'ları oluşturun, içlerinde Flexbox kullanın.

Grid + Flexbox Kombinasyonu 
.page-layout {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: auto 1fr auto;
  height: 100vh;
}

.header {
  grid-column: 1 / -1;
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.sidebar {
  grid-row: 2;
  display: flex;
  flex-direction: column;
}

.main {
  grid-row: 2;
  display: flex;
  flex-direction: column;
}

Önemli Not

Grid ve Flexbox'ı birlikte kullanmak en güçlü yaklaşımdır. Grid ile genel layout'u oluşturun, Flexbox ile component'ları düzenleyin.

Sonuç

CSS Grid ve Flexbox, farklı amaçlar için tasarlanmıştır. Doğru aracı doğru yerde kullanarak daha temiz ve maintainable kod yazabilirsiniz.

Özet

  • Grid: 2D layout'lar için
  • Flexbox: 1D layout'lar için
  • Birlikte kullanım mümkün
  • Her aracın kendine özgü avantajları var
  • Proje gereksinimlerine göre seçim yapın
