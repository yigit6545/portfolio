ES2024'te Gelen Yeni Özellikler

Frontend Developer Senior Developer
ES2024 JavaScript New Features

JavaScript'in en son versiyonu ES2024, geliştiricilere yeni araçlar ve özellikler sunuyor. Bu yazıda ES2024'te gelen en önemli yenilikleri inceleyeceğiz.

Array.prototype.toSorted()

Yeni toSorted() metodu, orijinal array'i değiştirmeden sıralanmış bir kopya döndürür.

toSorted() Örneği 
const numbers = [3, 1, 4, 1, 5];
const sorted = numbers.toSorted();
console.log(sorted); // [1, 1, 3, 4, 5]
console.log(numbers); // [3, 1, 4, 1, 5] - orijinal değişmedi

Array.prototype.toReversed()

toReversed() metodu, array'i ters çevirir ancak orijinal array'i değiştirmez.

toReversed() Örneği 
const fruits = ['apple', 'banana', 'cherry'];
const reversed = fruits.toReversed();
console.log(reversed); // ['cherry', 'banana', 'apple']
console.log(fruits); // ['apple', 'banana', 'cherry'] - orijinal değişmedi

Array.prototype.with()

with() metodu, belirtilen indeksteki elementi değiştirir ve yeni bir array döndürür.

with() Örneği 
const colors = ['red', 'green', 'blue'];
const newColors = colors.with(1, 'yellow');
console.log(newColors); // ['red', 'yellow', 'blue']
console.log(colors); // ['red', 'green', 'blue'] - orijinal değişmedi

Object.groupBy()

Object.groupBy() metodu, array elementlerini belirtilen fonksiyona göre gruplar.

Object.groupBy() Örneği 
const inventory = [
  { name: 'asparagus', type: 'vegetables', quantity: 5 },
  { name: 'bananas', type: 'fruit', quantity: 0 },
  { name: 'goat', type: 'meat', quantity: 23 },
  { name: 'cherries', type: 'fruit', quantity: 5 }
];

const grouped = Object.groupBy(inventory, ({ type }) => type);
console.log(grouped);
// {
//   vegetables: [{ name: 'asparagus', type: 'vegetables', quantity: 5 }],
//   fruit: [
//     { name: 'bananas', type: 'fruit', quantity: 0 },
//     { name: 'cherries', type: 'fruit', quantity: 5 }
//   ],
//   meat: [{ name: 'goat', type: 'meat', quantity: 23 }]
// }

Önemli Not

Bu özellikler henüz tüm tarayıcılarda desteklenmeyebilir. Production'da kullanmadan önce browser compatibility'yi kontrol edin.

Sonuç

ES2024, JavaScript'e güçlü yeni özellikler ekliyor. Bu özellikler özellikle immutable programming ve data manipulation konularında büyük kolaylık sağlıyor.

Özet

  • toSorted() - Immutable array sorting
  • toReversed() - Immutable array reversing
  • with() - Immutable array element replacement
  • Object.groupBy() - Array grouping
  • Daha iyi immutable programming desteği
