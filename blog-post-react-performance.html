React Performance Optimizasyonu: Best Practices

Frontend Developer Senior Developer
React Performance Optimization

React uygulamalarınızı optimize etmek, kullanıcı deneyimini önemli ölçüde iyileştirir. Bu yazıda React performance optimizasyonu için en etkili teknikleri inceleyeceğiz.

React.memo ile Component Memoization

React.memo, component'ların gereksiz re-render'larını önler. Props değişmediği sürece component yeniden render edilmez.

React.memo Örneği 
import React, { memo } from 'react';

const ExpensiveComponent = memo(({ data, onUpdate }) => {
  console.log('ExpensiveComponent rendered');
  
  return (
    <div>
      <h2>{data.title}</h2>
      <p>{data.description}</p>
      <button onClick={onUpdate}>Update</button>
    </div>
  );
});

export default ExpensiveComponent;

useMemo ile Hesaplama Memoization

useMemo, pahalı hesaplamaları cache'ler ve sadece dependency'ler değiştiğinde yeniden hesaplar.

useMemo Örneği 
import React, { useMemo } from 'react';

function ExpensiveCalculation({ items }) {
  const expensiveValue = useMemo(() => {
    console.log('Expensive calculation running...');
    return items.reduce((sum, item) => sum + item.value, 0);
  }, [items]);

  return (
    <div>
      <h2>Total: {expensiveValue}</h2>
    </div>
  );
}

useCallback ile Function Memoization

useCallback, function'ları memoize eder ve sadece dependency'ler değiştiğinde yeni function oluşturur.

useCallback Örneği 
import React, { useCallback, useState } from 'react';

function ParentComponent() {
  const [count, setCount] = useState(0);
  const [name, setName] = useState('');

  const handleClick = useCallback(() => {
    setCount(c => c + 1);
  }, []);

  return (
    <div>
      <input 
        value={name} 
        onChange={(e) => setName(e.target.value)} 
      />
      <ChildComponent onClick={handleClick} />
      <p>Count: {count}</p>
    </div>
  );
}

Lazy Loading ile Code Splitting

Lazy loading, component'ları ihtiyaç duyulduğunda yükler ve initial bundle size'ı küçültür.

Lazy Loading Örneği 
import React, { lazy, Suspense } from 'react';

const LazyComponent = lazy(() => import('./LazyComponent'));

function App() {
  return (
    <div>
      <h1>My App</h1>
      <Suspense fallback={<div>Loading...</div>}>
        <LazyComponent />
      </Suspense>
    </div>
  );
}

Önemli Not

Performance optimizasyonu yaparken önce problemi tespit edin. React DevTools Profiler ile hangi component'ların gereksiz render yaptığını görebilirsiniz.

Virtual Scrolling

Büyük listeler için virtual scrolling kullanarak sadece görünür elementleri render edebilirsiniz.

Performance İpuçları

  • React.memo ile component memoization
  • useMemo ile hesaplama memoization
  • useCallback ile function memoization
  • Lazy loading ile code splitting
  • Virtual scrolling büyük listeler için
  • React DevTools Profiler kullanın

Sonuç

React performance optimizasyonu, kullanıcı deneyimini önemli ölçüde iyileştirir. Doğru teknikleri kullanarak daha hızlı ve responsive uygulamalar geliştirebilirsiniz.

Özet

  • Memoization teknikleri kullanın
  • Code splitting ile bundle size'ı küçültün
  • Virtual scrolling büyük listeler için
  • React DevTools ile profil yapın
  • Gereksiz re-render'ları önleyin
